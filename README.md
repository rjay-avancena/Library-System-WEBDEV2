SMART LIBRARY WEB SYSTEM – USER GUIDE


1. SYSTEM OVERVIEW
The Smart Library Web System is a web-based application for managing library operations such
as borrowing, returning, reserving, and monitoring book status. The system supports four user
roles: Student, Teacher, Librarian, and Staff, each having specific permissions.

3. GETTING STARTED
How to Access the System
1. Open your web browser.
2. Go to: http://localhost/LIBRARYSYSTEM/
3. Choose Login or Sign Up depending on your status.

   
IF YOU ARE A NEW USER (SIGN UP)
1. Click Sign Up.
2. Fill out the required information:
o First Name
o Last Name
o Select Role (Student, Teacher, Librarian, Staff)
o Email
o Password
o Contact Number
3. Click Sign Up.

   
IF YOU ALREADY HAVE AN ACCOUNT (LOGIN)
1. Enter your email and password.
2. Click Login.
3. You will be redirected to your dashboard based on your role.

   
3. USER ROLES
1. Student
• Can borrow up to 3 books per semester
• Must return 3 books for clearance
• Can reserve books online
• Can view your reserved books
• Can view your penalties
• Can request a clearance




3. Teacher
• Can borrow unlimited books
• Must return all the books you borrowed at the end of the school year for clearance
• Can reserve books online
• Can view your reserved books
• Can view your penalties
• Can request a clearance




5. Librarian
• Can add, update, and archive books
• Can unarchive books
• Manages book inventory and book details




7. Staff
• Facilitates borrowing by inputting a Username, Book Title, and Due Date, then pressing
Facilitate Borrow
• Facilitates returning by inputting the Username and the Book Title that he/she
borrowed
• Can view borrower status by typing the Username and the Book Title, and can see
his/her penalties
• Assists students and teachers during clearance by inputting Username, School Year, and
setting the clearance status (cleared or not)




4. FEATURES BY ROLE
   
A. Student Features
Borrow a Book
1. Go to Borrow Book.
2. Click Borrow Book.
3. Pick the book you want and set the date.
Rule: Max 3 books per semester.
Return a Book
1. Go to Return Book.
2. Click Return Book.
3. You can see the books you borrowed and you can return them.
Reserve a Book
1. Go to Reserve Book.
2. Click Reserve Book.
3. View the available books that you want to reserve and set the date.
View Borrowing Status
1. Go to View Reservations.
2. There you can see the books you reserved.
View Penalties
1. Go to View Penalties.
2. There you can see all the penalties you have committed.
Clearance
1. Go to Clearance.
2. Input the academic year and request a clearance.

   
   
B. Teacher Features
Borrow a Book
1. Go to Borrow Book.
2. Click Borrow Book.
3. Choose the book you want and set the due date.
Rule: Teachers can borrow unlimited books.
Return a Book
1. Go to Return Book.
2. Click Return Book.
3. You will see the list of books you borrowed and select which one to return.
Note: All borrowed books must be returned at the end of the school year for
clearance.
Reserve a Book
1. Go to Reserve Book.
2. Click Reserve Book.
3. View the available books you want to reserve and set the date.
View Borrowing Status
1. Go to View Reservations.
2. You can see all the books you reserved and their status.
View Penalties
1. Go to View Penalties.
2. You can see all penalties you have.
Clearance
1. Go to Clearance.
2. Input the academic year.
3. Request a clearance.

   

C. Librarian Features
Add a Book
1. Click Add Book.
2. Fill in the book details:
(Title, Author, ISBN, Category ID, Publisher, Book Price, Total Copies, Copies Available,
Status)
3. Click Add Book.
Edit or Update a Book
1. Click Update Book.
2. Then press Edit. Here you can edit all the book details:
(Title, Author, ISBN, Category ID, Publisher, Book Price, Total Copies, Copies Available,
Status)
3. Click Update Book.
Inventory Book
1. Click Inventory.
2. You can see all the book information in the library.
Archive a Book
1. Go to Archive Book.
2. Click Archive on the book you want to archive.
View Archived Books
1. Click Archived Books.
2. You can see the books you archived.
3. Press Restore to unarchive a book, then go back to Archive Book to view the book you
just restored.



D. Staff Features
Facilitate Borrowing
1. Search for the username.
2. Type the book title.
3. Set the due date.
4. Click Facilitate Borrow to successfully borrow a book for a person.
Facilitate Return
1. Open Facilitate Return.
2. Input the username and the book title.
3. Confirm return.
View Borrower Status
• Input the username to see:
• Active borrowed books
• Unpaid penalties
Manage Clearance
You can manage people by entering the User Name, Academic Year, and Set Status (optional)
to fully clear or not clear their clearance.
